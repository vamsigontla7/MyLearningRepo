Building the home town app
1. Add a <meta> view port to the page with intial scale set
2. Adjust css and markup so that everything displays in single column. Use relative widths so that the things stretch to fit across any viewport width.
3. Make sure your touch targets are easy to hit.
4. Test ypur site across different view ports. Try on different phones, tablets and desktops.
